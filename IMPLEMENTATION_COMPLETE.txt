WHAT'S IMPLEMENTED
================================================================================

1. ERROR DETECTION (result_classifier.py)
   - Identifies error_max_turns in Claude Code results
   - Returns structured metadata instead of failure signal
   - Enables graceful error handling in orchestrator

2. AUTO-REMEDIATION (auto_remediate.py)
   - Detects max_turns errors in task queue
   - Retry strategy: escalates on repeated failures
   - Model escalation: Haiku → Sonnet → Opus on cap
   - Injects focused implementation directives for difficult tasks

3. LLM CALL GATING (model_policy.py + runner.py)
   - Skips expensive LLM verification for low-risk diffs
   - Only skips when BOTH conditions met:
     * Tests pass
     * Build passes
   - NEVER skips when:
     * High blast radius
     * High-risk flag set
     * Constitutional files touched (auth, security, compliance, etc.)
     * Tests or build failed
   - Configurable policy with sensible defaults
