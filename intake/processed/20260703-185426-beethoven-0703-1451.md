  proof: `python -m pytest runner/tests/ -q` exits 0 (incl. new runner/tests/test_build_cache.py)
  prompt: |
    The merge gate runs a full app build (nuxt build) per task — the dominant cost.
    Add runner/build_cache.py computing a cache key from each worktree's lockfile(s)
