    message across channels (call, voicemail, SMS, WhatsApp, email, Zoom recording).

    Steps:
    - Inspect the repo first; match existing conventions (TS types location, test runner,

      every channel enum, both directions).
    - Export from a single module other tasks can import. Do NOT add DB tables here.
    - If the repo's typecheck command differs (e.g. vue-tsc), use that and note it in the PR.
